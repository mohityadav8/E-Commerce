# 🛒 E-Commerce Website — Frontend Project

> A modern and responsive shopping website built using **HTML, CSS and JavaScript**.
> Designed to simulate a real online shopping experience including product browsing, cart interaction and UI animations.

## 📌 Features

✨ Clean modern UI
✨ Fully responsive design (Mobile + Tablet + Desktop)
✨ Product listing & product preview pages
✨ Add to cart interaction
✨ Navigation bar & interactive buttons
✨ Hero banners & promotional sections
✨ Smooth scrolling & hover animations
✨ Organized layout similar to real shopping websites


```bash

```

3. Run
